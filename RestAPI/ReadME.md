User Management REST API

A RESTful API built with Python, Flask, Flask-RESTful, and SQLAlchemy.
This project demonstrates CRUD operations, database modeling, ORM usage, and structured backend development.

Technologies Used
Python
Flask
Flask-RESTful
SQLAlchemy
Postman (API testing)
This project implements a simple User Management API that allows:
Creating users,Retrieving users,Updating users,Deleting users
The application uses SQLAlchemy ORM to map Python classes to database tables and manage database operations safely.
