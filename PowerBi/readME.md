This project visualizes monthly credit card spending to provide actionable insights into expenses across different categories and dates. The goal was to create an interactive and dynamic dashboard that helps track spending patterns and identify trends.

Key Highlights:

Built a clean data model and created calculated columns to extract the Day from the date field.

Developed DAX measures for SUM, AVERAGE, and other KPIs, using CALCULATE() to filter and analyze data dynamically.

Designed interactive visuals and slicers to explore expenses by category, date, and other dimensions.

Implemented a drill-through feature to allow detailed analysis of individual categories, giving a clear overview of spending patterns.

Derived insights such as top spending categories and high-spending days, enabling better financial decision-making.

Tools Used: Power BI Desktop, DAX, Excel (for initial data preparation).
